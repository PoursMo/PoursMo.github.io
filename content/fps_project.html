--- layout: default title: Unnamed FPS Project ---

After finishing my course on the Fundamentals of Unity & C#, I started working on a first person puzzle game with some inspiration from the Portal games.
In this game, the player has to use a ball that can be picked up and thrown using his gun. After the ball is thrown, the player can teleport at the position of the ball. This is used to solve puzzles to reach the end of the level.

I created different material types for the walls : blue is sticky so when the ball touches it, it stops instantly. Green is bouncy so can be used to throw the ball in places that wouldn't be accessible otherwise.

I also added different types of interactable objects : The ball itself is interactable at the beginning of the level so that the player can pick it up.
A button that can be pressed to trigger any function using Unity Event system, such as opening a door.
A vessel where the player would put the ball to open the last door of the level.

I learned to use Pro Builder to build the level of the game, and the basics of Blender to create the vessel.
