This is my most recent project. The idea is to create a roguelite (a Roguelike with progression elements that persists even after failure) based on melee combat.

My objective for the game is to have the player progress through procedurally generated levels and find power-ups that will make them stronger for the current run they're on, as well as persisting elements like permanent bonuses.
I also want to add a boss fight with telegraphed attacks, and make the combat system simple, reactive, and rewarding skill (like a Soulslike)

Feel free to go check out the subcategories of this project to see what I worked on so far.
