The idea for this game was to create a roguelite (a Roguelike with progression elements that persists even after failure) based on melee combat.

I implemented an algorithm to procedurally generate dungeons that the player would go through, and started working on a combat system that is simple, reactive, and one that rewards skill (like a Soulslike).
