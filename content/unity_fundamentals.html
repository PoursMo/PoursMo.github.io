--- layout: default title: Unity & C# Fundamentals ---

I followed a course which purpose was to teach the fundamentals of Unity and C#. The course was divided in 3 projects described below.
I already had some knowledge in both Unity and C# when I followed this course, which allowed me to finish the first 2 projects quickly.

First Project

The first project was using Unity's 2D Game Kit. It's goal is to get people used to Unity and learn the main components of it, namely the different windows.
No coding was done for this project, only placing GameObject and editing properties in the Inspector.

Second Project

The second project was done in Visual Studio only with .NET SDK. The objective was to create the Hangman game using C#, with the possiblity to replay, and tracking the letters the player already tried. I added 2 different gamemodes, single player where the word to guess would be a randomly chosen word from a list, and multiplayer where the first player would type in the word for the second player to guess. The hangman is drawn using ASCII art.

Third Project

The third and last project's goal was to put into practice the knowledge acquired in the previous projects, while still learning new things in both C# and Unity
